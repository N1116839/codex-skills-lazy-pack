# GDP HTML Checklist

- Confirm every source file has a database record.
- Confirm the database can answer short queries: transport, storage location, supplier, retention, temperature, recall, deviation, CAPA.
- Confirm removed content is removed everywhere: cold-chain claims, cold storage, unsupported temperature systems.
- Confirm repeated UI sections are not duplicated across top learning points and lower panels.
- Confirm document-map items are clickable when SOP summaries exist.
- Confirm Q&A keeps questions without requiring a user name.
- Confirm mobile filters use buttons or a tested custom selector, not an unreliable native select.
- Confirm rankings do not show user leaderboard if the user asked to remove it.
